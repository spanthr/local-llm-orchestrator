================================================================================
OLLAMA LOCAL LLM - REAL USAGE EXAMPLES
================================================================================

All commands use: run_task.bat <task> "<prompt>"
Or with Python: python cli.py --task <task> --prompt "<prompt>"

================================================================================
TASK 1: HOLIDAY PLANNER
================================================================================

Example 1:
  run_task.bat task1 "Plan a 5-day trip to Paris. Budget: $3000. What are the top 10 attractions?"

  Expected: Hotel suggestions, daily itinerary, budget breakdown, restaurant recommendations

Example 2:
  run_task.bat task1 "Create a beach vacation checklist for Bali. What should I pack? Where to stay?"

  Expected: Packing list, best beaches, accommodation options, local tips

Example 3:
  run_task.bat task1 "Plan a winter ski trip to Switzerland. Difficulty level: intermediate. Best resorts?"

  Expected: Ski resorts ranked, accommodation, costs, gear recommendations, travel tips

Example 4:
  run_task.bat task1 "Weekend trip ideas in California. I like hiking and food. Suggest 3 options."

  Expected: 3 destinations with hiking trails, local restaurants, distance from major cities

================================================================================
RESPONSE TIME
================================================================================

These are LOCAL models on your machine:
  • mistral: ~10-30 seconds for detailed responses
  • neural-chat: ~5-15 seconds
  • phi: ~2-5 seconds (but less detailed)

First time downloading a model: 5-10 minutes depending on internet

================================================================================
CUSTOMIZATION
================================================================================

To add your own tasks:
1. Open cli.py
2. Add to TASKS dictionary:
   "task4": {"name": "Your Task", "model": "mistral"}
3. Save and use: run_task.bat task4 "Your prompt"

To use different models:
1. List available: run_task.bat list
2. Download more: ollama pull llama2
3. Update cli.py task config

================================================================================
PERFORMANCE TIPS
================================================================================

1. First run of a model is slower (model loading from disk)
2. Longer prompts = longer response time
3. GPU acceleration: Ollama auto-detects RTX 2060
4. Increase speed: Use phi model instead of mistral
5. Increase quality: Use mistral instead of neural-chat
