================================================================================
BATCH SCRIPT COMMANDS (Windows - Easiest)
================================================================================

Syntax: run_task.bat <command> [options]

TASK COMMANDS:
  run_task.bat task1 "Your prompt"     Run Holiday Planner task
  run_task.bat task2 "Your prompt"     Run ROS2/CARLA Development task
  run_task.bat task3 "Your prompt"     Run Media Editing task

MANAGEMENT COMMANDS:
  run_task.bat list                    List all installed models
  run_task.bat verify                  Check setup and privacy settings

EXAMPLES:
  run_task.bat task1 "Plan a trip to Tokyo"
  run_task.bat task2 "Generate PID controller code"
  run_task.bat task3 "Color grading tips for portraits"

================================================================================
PYTHON CLI COMMANDS (Direct)
================================================================================

Syntax: python cli.py [command] [options]

TASK COMMANDS:
  python cli.py --task task1 --prompt "Your prompt"
  python cli.py --task task2 --prompt "Your prompt"
  python cli.py --task task3 --prompt "Your prompt"

MANAGEMENT COMMANDS:
  python cli.py --verify               Check setup and privacy
  python cli.py --list-models          Show installed models
  python cli.py                        Show help

EXAMPLES:
  python cli.py --task task1 --prompt "Beach vacation ideas"
  python cli.py --task task2 --prompt "Implement sensor fusion"
  python cli.py --task task3 --prompt "Lighting setup for video"

================================================================================
OLLAMA COMMANDS (Advanced)
================================================================================

These work directly with Ollama:

MODEL MANAGEMENT:
  ollama list                          List all installed models
  ollama pull mistral                  Download mistral model
  ollama pull neural-chat              Download neural-chat model
  ollama pull llama2                   Download llama2 model
  ollama remove mistral                Delete mistral model
  ollama show mistral                  Show model info

RUNNING MODELS:
  ollama run mistral "Your prompt"     Run mistral directly
  ollama run neural-chat "Your prompt" Run neural-chat directly
  ollama serve                         Start Ollama server (usually auto)

CHECKING STATUS:
  ollama --version                     Show Ollama version
  ollama list                          Show all models + size

================================================================================
ENVIRONMENT VARIABLES
================================================================================

Set these for customization:

OLLAMA_MODELS
  Path to models directory
  Default: E:\llm_workspace\ollama_models
  Set with: setx OLLAMA_MODELS "E:\llm_workspace\ollama_models"

OLLAMA_TELEMETRY
  Enable/disable usage tracking
  Value: false (for privacy)
  Set with: setx OLLAMA_TELEMETRY false

OLLAMA_HOST
  Server address
  Default: localhost:11434 (local only)
  Can't change without breaking local-only guarantee

================================================================================
TROUBLESHOOTING COMMANDS
================================================================================

PROBLEM: "Model not found"
SOLUTION: Download it
  ollama pull mistral

PROBLEM: "Out of disk space"
SOLUTION: Remove unused models
  ollama remove phi
  Check space: dir E:\

PROBLEM: "Response is slow"
SOLUTION: Use faster model
  run_task.bat task1 "prompt"  (tries mistral first)
  Edit cli.py to use phi instead

PROBLEM: "Ollama not running"
SOLUTION: Restart Ollama
  On Windows, usually auto-starts
  Manually: ollama serve

PROBLEM: "Permission denied on setup_env.bat"
SOLUTION: Run as Administrator
  Right-click Command Prompt → "Run as administrator"
  Then: setup_env.bat

PROBLEM: "Command not found"
SOLUTION: Check PATH or use full path
  Full path: "E:\llm_workspace\run_task.bat" task1 "prompt"

================================================================================
ADVANCED: EDITING TASKS
================================================================================

Add custom task:
1. Open E:\llm_workspace\cli.py in text editor
2. Find TASKS = { ... }
3. Add:
   "task4": {"name": "My Task", "model": "mistral"}
4. Save
5. Use: run_task.bat task4 "Your prompt"

Change model for task:
1. Edit TASKS in cli.py
2. Change model from "mistral" to "neural-chat" or "phi"
3. Save
4. Rerun task

Add more models:
1. ollama pull llama2
2. Edit cli.py to use it
3. Save and run
